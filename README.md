## üöÄ Getting Started (v0.1 Simulation)

This repository contains the genesis code for the Laniakea Protocol. You can run the simulation of its core concepts right in your browser.

1.  **Open in Codespaces:** Click the "Code" button above, go to the "Codespaces" tab, and create a new codespace. This will set up a complete development environment for you automatically.
2.  **Activate Virtual Environment:** Once the codespace loads, open a terminal and run:
    ```bash
    source venv/bin/activate
    ```
3.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Run the Cosmic Simulation:**
    ```bash
    python3 main.py
    ```
